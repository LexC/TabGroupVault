# Changelog

## 1.1.0

- Add an optional `tab-groups.csv` report checkbox, off by default.
- Change `tab-groups.csv` to include selected pages only with a simplified column order.
- Add numbered/page-title filename modes with sanitized title filenames.
- Merge destination, report, filename, and conflict controls into one compact Output section.
- Remember export settings and best-effort selected output folder access.
- Add selected-by-default group and tab toggles with compact selected-tab numbering.
- Move HTTP/HTTPS page access to runtime optional host permissions for HTML exports.
- Add current-window counts to the popup.
- Replace `TESTING.md` with focused manual files plus unit, Playwright, and CI coverage.

## 1.0.0

- Initial store-ready release.
- Export grouped tabs as HTML snapshots, HTML asset folders, MHTML archives, or CSV indexes.
- Support selected-folder export with an explicit Downloads fallback.
- Add store packaging structure, validation scripts, and release ZIP generation.
